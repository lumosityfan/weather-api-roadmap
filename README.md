# weather-api-roadmap
Weather API project from Roadmap.sh

https://roadmap.sh/projects/weather-api-wrapper-service

To run the project, simply do

```bash
python index.py
```

and click on the localhost link in the terminal that the program provides.

You need to provide various environment variables before this program can work fully.

VisualCrossing API Key
Redis Database Host
Redis Database Port
